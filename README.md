# 📊 Sales Insights Dashboard

A complete data analysis project using Python, Pandas, NumPy, and Matplotlib to generate synthetic sales data, analyze trends, and create an executive dashboard.

## 🚀 Features
- **Data Generation**: Creates 5,000 realistic sales records using NumPy
- **Data Analysis**: Calculates total revenue, average order value, top products, and monthly trends
- **Visualization**: Generates a 4-panel executive dashboard (line chart, bar chart, histogram, pie chart)
- **Professional Structure**: Modular code with error handling, logging, and command-line arguments

## 🛠️ Tech Stack
- **Python 3.8+**
- **Pandas** - Data manipulation
- **NumPy** - Numerical operations
- **Matplotlib** - Data visualization
